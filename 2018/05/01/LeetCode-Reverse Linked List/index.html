206. Reverse Linked List

题目描述: 给定一个链表的头结点head，反转整个链表。

解决思路：

  1. 初始化一个前向结点指针为空 (NULL)，并开始循环

  2. 存储当前结点指针指向的后一个结点

  3. 让当前结点指向前向结点

  4. 让前向结点等于当前结点

  5. 让当前结点等于之前存储的后向结点

  6. 当前结点为空时停止循环
